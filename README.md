# 🪐 Solar System UI

An interactive solar system built with pure HTML & CSS — planets, orbit animations, speed controls and a view mode. Zero JavaScript. Just CSS magic. ✨

- ⚡ Animation speed controls
- 🔭 View mode toggle
- 💫 Pure CSS animations
